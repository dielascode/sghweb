Layanan - Magang

Program Magang di Smart Green House merupakan kesempatan bagi mahasiswa untuk memperoleh pengalaman langsung dalam penerapan teknologi pertanian modern berbasis Internet of Things (IoT). Melalui kegiatan ini, peserta akan belajar bagaimana sistem otomatisasi digunakan untuk memantau kondisi lingkungan, mengatur penyiraman tanaman, serta mengoptimalkan pertumbuhan tanaman secara berkelanjutan. Selama program berlangsung, peserta magang akan terlibat dalam berbagai aktivitas praktis seperti:

  • Instalasi dan kalibrasi sensor suhu, kelembapan, dan intensitas cahaya.
  • Pemrograman mikrokontroler dan integrasi data ke sistem monitoring berbasis web.

Selain keterampilan teknis, program magang juga menanamkan nilai kolaboratif dan problem solving, di mana mahasiswa diajak bekerja dalam tim lintas jurusan — seperti teknologi informasi, pertanian, dan teknik elektro — untuk menciptakan solusi nyata dalam dunia pertanian digital.
